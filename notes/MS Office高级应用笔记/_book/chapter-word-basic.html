1.1 快捷键

Table 1.1: word中的常用快捷键
快捷键 功能
Ctrl + D 字体
Ctrl + F 查找
Ctrl + G 定位
Ctrl + H 替换
Ctrl + K 插入超链接
Ctrl + N 新建
Ctrl + O 打开文件
Ctrl + P 打印
Ctrl + W 只关闭文件不关闭 Word 程序1
Alt + = 插入公式
F4 重复操作2
Tab 变为下一级标题（降级）
Shift + Tab 变为上一级标题（升级）
Shift + F5 将光标移到上次编辑的位置
Ctrl + Shift + F9 将域转变为纯文本
选择文本
鼠标双击3 选择一个单词
在段落的左侧空白处双击鼠标 选中一个段落
三击鼠标 选中一个段落
Ctrl + 单击 选中一个句子
Home/End 光标移动到行头/行尾
Ctrl + Home/End 光标移动到全文头/全文尾
Shift + Home/End 选中从光标到行头/行尾
Ctrl + Shift + Home/End 选中从光标到全文头/全文尾
Alt + 拖曳 垂直选中一块文本4
F8 开启选择模式，以光标为选择的基点5；ESC取消
复制粘贴
Ctrl + Shift + C/V 复制粘贴格式

1.2 一些技巧

  1. 多样化粘贴，可以粘贴为链接，保持与数据源的更新能力
  2. 删除不连续的空行：将两个回车^p替换为一个回车^p

  1. Word 程序打开很耗时，如果只是要关闭一个文件，仍要继续使用 Word 处理其他文件，最好用这个快捷键。↩︎

  2. 在excel中尤其好用！↩︎

  3. 或：光标在词头，按Ctrl + Shift + Right；光标在词尾，按Ctrl + Shift + Left。↩︎

  4. 很多行文字有相同的前缀、后缀和中间部分，当你想编辑的时候，可以垂直选中、统一修改。↩︎

  5. 按1次F8开启选择模式，再按1次选中当前单词，2次选中句子，3次选中段落，4次选中全文。↩︎
