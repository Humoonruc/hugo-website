## 数据清洗

宽数据转换为长数据，可以使用Excel中的逆数据透视表：



1. 【数据】-【来自表格/区域】-【创建表】，勾选【表包含标题】
2. Power Query 编辑器自动弹出
3. 在其中先选中要保留的列，然后【转换】-【逆透视列】-【逆透视其他列】，即可完成转换
